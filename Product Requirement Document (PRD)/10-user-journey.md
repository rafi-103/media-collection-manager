# User Journey (Alex)

1. **Login/Open App:** Alex opens the Personal Media Collection Manager.
2. **View Dashboard:** Alex sees a dashboard showing total items (e.g., 50 Books, 30 Movies).
3. **Add a New Book:** Alex clicks "Add Item", selects "Book", fills in title "Harry Potter", author "J.K. Rowling", status "Currently Reading", and clicks "Save".
4. **Search:** Alex uses the search bar to check if he owns "Lord of the Rings" to avoid buying a duplicate.
5. **Lend an Item:** Alex finds a book, clicks "Lend", enters friend's name "Mike", sets return date, and saves.
6. **View Movie Details:** Alex clicks a movie card to see its details (director, rating, streaming platform).
7. **Logout:** Alex logs out.

**Outcome:** Alex now has a complete record of his collection and lent items.
