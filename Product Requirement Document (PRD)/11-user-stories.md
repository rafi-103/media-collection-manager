# User Stories

1. **As a user,** I want to add a new book to my collection so that I can track my reading progress.
2. **As a user,** I want to add a new movie to my collection so that I remember if I have watched it.
3. **As a user,** I want to search for a specific title so that I can check if I already own it.
4. **As a reader,** I want to mark a book as "Loaned" to a specific friend so that I don't forget who borrowed it.
5. **As a movie fan,** I want to mark a movie as "⭐ Favorite" so that I can find my top films easily.
6. **As a user,** I want to edit item details (e.g., change status from "Reading" to "Completed") so that my collection stays up-to-date.
7. **As a user,** I want to delete an item so that I can remove items I no longer own.
8. **As a user,** I want to filter my collection by "Book" or "Movie" so that I only see one type at a time.
9. **As a user,** I want to see a dashboard with collection statistics so that I know my overall progress.
