# Technical Design Document (TDD)

**Overview:**
This document describes the technical architecture and design decisions for the Personal Media Collection Manager.

**Technology Stack:**
- **Frontend:** React (for building UI components), React Router (for navigation), Axios (for API calls).
- **Backend:** FastAPI (for creating RESTful endpoints), SQLAlchemy (ORM), Pydantic (for data validation).
- **Database:** PostgreSQL (relational database).
- **Version Control:** Git/GitHub with Issues and Kanban Board.
