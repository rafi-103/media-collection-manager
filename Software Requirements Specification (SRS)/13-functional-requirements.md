# Functional Requirements

The system shall provide the following functions:

**FR-01: Add Media Item**
- User can add a new item by providing Title, Type (Book/Movie), Creator (Author/Director), Genre, Status, and Rating.

**FR-02: View All Items**
- User can view all items in a grid/dashboard format.

**FR-03: Edit Media Item**
- User can edit the details of any existing item.

**FR-04: Delete Media Item**
- User can delete an item (with confirmation prompt).

**FR-05: Search Items**
- User can search for items by Title or Creator using a keyword search.

**FR-06: Filter Items**
- User can filter items by Type (Book/Movie) and Status.

**FR-07: Dashboard Stats**
- System shall display total items, and breakdown by type.

**FR-08: Lend Book**
- For books, user can mark an item as "Loaned" to a friend with a return date.

**FR-09: Movie Favorites**
- For movies, user can mark an item as "⭐ Favorite".
- For movies, user can log the streaming platform or format.
