# Product Requirement Document (PRD)

**Product Name:** Personal Media Collection Manager

**Objective:**
To provide a web-based solution for users to catalog, organize, and track their personal collections of books and movies.

**Core Features (MVP):**
1. **Add Item:** Users can add a new book or movie with details (Title, Creator, Genre, Status, Rating).
2. **View Items:** Display all items in a grid/dashboard view.
3. **Edit Item:** Users can update item details.
4. **Delete Item:** Users can remove items from the collection.
5. **Search:** Keyword search across titles and creators.
6. **Filter:** Filter by Type (Book/Movie) and Status.
7. **Dashboard Stats:** Show total items and breakdown by category.

**Book-Specific Features:**
- Track reading status (Want to Read, Currently Reading, Completed).
- Lend to a friend (Name, Return Date).

**Movie-Specific Features:**
- Track viewing status (Watchlist, Currently Watching, Watched).
- Mark as "⭐ Favorite".
- Log streaming platform or format (Netflix, Blu-ray, etc.).
