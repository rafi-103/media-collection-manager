# Project Overview

**Project Name:** Personal Media Collection Manager

**Description:**
The Personal Media Collection Manager is a web application designed to help users organize, track, and manage their personal collections of both books and movies. Many people forget which books they already own, accidentally buy duplicate movies they have already watched, or lend physical copies to friends and cannot remember who borrowed them. This app solves that problem by allowing users to easily add, view, edit, search, and delete items from their unified collection.

**Goals:**
- Provide a centralized platform to catalog personal media.
- Allow users to track reading/watching progress.
- Enable lending tracking for physical books.
- Offer filtering and search for quick access.

**Scope:**
The application will be built using React (Frontend), FastAPI (Backend), and PostgreSQL (Database), covering full CRUD operations. This is a solo project, so the scope is focused on core functionality with optional bonus features.
